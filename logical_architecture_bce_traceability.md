Statuso taisyklė: failų keliai, React route'ai, `api.*` kvietimai, Laravel route -> controller mapping, controller/service/model importai ir modelių migracijos šiame faile laikomi VERIFIED, nes jie nustatyti iš repo. Loginis sričių grupavimas ir dalis išplėstinių flow service grandinių laikomi INFERRED, kai jie remiasi domeniniu pavadinimu arba service-to-service kontekstu, o ne tiesioginiu page -> API -> route -> controller importu.
