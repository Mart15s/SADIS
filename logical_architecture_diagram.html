Stereotypes: React pages, route guards, AuthContext ir Client API yra pagrindiniai naudotojo sąsajos / API kliento «boundary» elementai. Laravel controlleriai ir service klasės yra «control» elementai. Eloquent modeliai yra «entity» elementai.
